Browser Command Panel - Your Personal Browser Assistant

How to Use

To activate the command panel, use the keyboard shortcut Ctrl+Shift+Space (or Command+Shift+Space on Mac).

You can also use the omnibox by typing "cmd" followed by a space in the address bar.

Available Commands

Command Description
history Open browser history
settings Open browser settings
clchistory Clear browsing history
cookies Open cookie settings
clccookies Clear all cookies
downloads Open downloads page
extensions Open extensions page
bookmarks Open bookmarks page
yt [query] Search YouTube
inst [query] Search Instagram
git [query] Search GitHub
help Open this help page

For any other input, the extension will perform a Google search with the provided query.
